# Online Quiz Maker

## Description

Online Quiz Maker is a simple web application that allows users to answer multiple-choice questions and view their final score at the end of the quiz.

## Features

* Display quiz questions dynamically
* Multiple-choice answer options
* Next button to move between questions
* Automatic score calculation
* Final score display

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

```
Online-Quiz-Maker/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html` or open it using a web browser.
4. Start answering the quiz questions.
